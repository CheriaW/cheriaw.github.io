Tuscany Villa

Tuscany Villa is a project showing my best attempt at replicating a mockup website. At the bottom of this website there's a section for guests to leave messages to the hotel (Tuscany Villa) and I struggled with getting the comment box to fit in the right 6 rows of the screen.

Organize Your Wardrobe

This website was designed to allow users to virtually organize their closet by category, color, season and occasion. Through database first programming users were given a list of seasons, colors, occassions and types of clothing to chose from when entering their clothes. In addition, users are given the ability to add their own special occassion that isn't listed in the given set. Users are also able to edit and delete items they've added to their closet.

Skills used:
Database first MVC

AQA Commute (Capstone Project for WCCI)

AQA Commute is a web app that shows users how many pounds of CO2 they generate with every commute. From there users can see what difference switching from personal transit to public transit will make in terms of CO2 emissions. The app offers insight for other factors as far as how their impact effects the rest of the world, why it matters, and other ways they can make a difference in their CO2 footprint (sources provided). Because this was a group project as well as a Capstone project, it was lengthier and more involved than projects I've done on my own in the past. I've learned a lot in this project not only about coding but also what it's like to work with a group in a scrum team.

Skills used:
C# | JavaScript | JQuery | Ajax | Bootstrap | Google Maps API | MVC | ASP.NET | MS SQL | Razor | HTML / CSS
